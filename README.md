## EGO Project

#### 1. Clone project

```bash
git clone https://github.com/huynhducthanhtuan/EGO-Project.git
```

#### 2. Run source backend

```bash
cd backend
```
```bash
npm install
```
```bash
npm start
```

#### 3. Run source frontend

```bash
cd frontend
```
```bash
npm install
```
```bash
npm start
```

#### 4. Run by using Docker

```bash
docker-compose build
```
```bash
docker-compose up -d
```
